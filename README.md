## 📅 Day 1: Environment Setup & The First Compile
**Date: May 25, 2026**

### 💻 The Journey Begins
* Today marked the official start of my C programming journey! 
* Learning about variables, data types, and basic syntax was smooth, but setting up the environment turned into a wild ride.

### 🛠️ Roadblocks & Troubleshooting (MinGW Setup)
* Hit a massive roadblock right at the start. 
* Spent nearly 5 hours just trying to find `gcc.exe` because Windows had hidden it for security reasons. 
* After a ton of frustration and checking file structures through the terminal, I finally got the compiler to respond. 

### 📁 Terminal Navigation & Folder Mismatch
* But the universe wasn't done with me yet. 
* Had a total "classic developer" moment when my next program refused to run. 
* After tracing my steps, I realized I had accidentally nested a folder inside another folder with the exact same name, completely throwing off my terminal's working directory! 📁🔄
* Had to learn the `cd ..` command on the fly, step back into the parent folder, remap the path, and finally—my code compiled and executed successfully! 

### 🧠 First Day Takeaway
> "Coding is probably just 10% of the job; the other 90% is debugging, fighting with terminal paths, and having ultimate patience. But that feeling when the code finally runs? Unmatched. Absolutely pumped for Day 2! 💻🔥"
